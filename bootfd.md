# Command: bootfd

### Starts microdisc bootsector

## SYNOPSYS
+ bootfd

## EXAMPLES
+ bootfd

Boots only microdisc boot sector with the help of microdisc.rom. In this version, we don't verify if a dsk is inserted or not. If there is no disk, the oric will crash
