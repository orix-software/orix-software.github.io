    XSCROH (Scrollup text mode)

    Descroption

    Scroll all lines from bottom to the top

    Examples

        .include "telestrat.inc"
    ldx     #$01 ; First line to scroll
    ldy     #26  ; Last line to scroll
    BRK_TELEMON XSCROH
    rts

    See scroll macro from orix-sdk to use it
