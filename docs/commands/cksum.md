# Command: cksum

### checksum utility

## SYNOPSYS
+ cksum -h|-v
+ cksum file [...]
+ cksum @batchfile

## EXAMPLES
+ cksum /bin/cksum

## DESCRIPTION
checksum and count the bytes in a file

## OPTIONS
* -h
                show this help message and exit
* -v
                display program version and exit

## SOURCE
https://github.com/orix-software/cksum
