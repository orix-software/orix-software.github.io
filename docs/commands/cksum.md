# Command: cksum

### checksum utility

## SYNOPSYS
+ cksum file [...]
+ cksum @batchfile
+ cksum -h

## EXAMPLES
+ cksum /bin/cksum

## DESCRIPTION
checksum and count the bytes in a file

## OPTIONS
*  -h
                show this help message and exit

## SOURCE
https://github.com/orix-software/cksum
