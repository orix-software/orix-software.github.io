# Command: more

### more utility

## SYNOPSYS
+ more <file>

## EXAMPLES
+ more /etc/autoboot.sub

## DESCRIPTION
**more** display a text file one screenfull at a time.

*[RETURN]* display next line.

*[CTRL]+C* break.

*q * quit.

Any other keys: display next page.
