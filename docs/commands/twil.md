# Command: twil

### Displays manage twilighte board

## SYNOPSYS
+ #twil -f  : displays firmware
+ #twil -w  : swap banks 4, 3, 2, 1 to ROM set
+ #twil -r  : swap banks 4, 3, 2, 1 to ROM set
+ #twil -sX : swap set for banks 4, 3, 2, 1
