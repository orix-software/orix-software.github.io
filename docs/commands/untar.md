# Command: untar

### untar utility

## SYNOPSYS
+ untar -t[v]f tarfile
+ untar -x[v]f tarfile

## EXAMPLES
+ untar -tf /home/test.tar

## DESCRIPTION
**untar** list and extract files in a TAR archive

## OPTIONS
*  -h ^Bshow this help message and exit
*  -t ^Blist files
*  -x ^Bextract files
*  -v ^Bverbose mode

## SOURCE
https://github.com/orix-software/untar
