* XMALLOC now tries to optimize the available memory when we try to get size equal or less which is available in a free memory slot
* Bugfix with fclose, when 2 files are opened, and the second file is closed, it close the first file instead.
* Bugfix when a file is not found, the file pointer is not allocated
