    #define VAR_SYS_BASE_INFO 0x20 / * Informations de base du systme actuel * /
/ * Le bit 6 est utilis pour indiquer la sous-classe du priphrique de stockage USB SubClass-Code, le bit 6 est 0 pour indiquer que la sous-classe est 6 et le bit 6 est 1 pour indiquer que la sous-classe est diffrente de 6 * /
/ * Le bit 5 est utilis pour indiquer l'tat de la configuration USB en mode priphrique USB et l'tat de la connexion du priphrique USB en mode hte USB * /
/ * En mode priphrique USB, si le bit 5 est 1, la configuration USB est termine et les bits 5 et 0 ne sont pas configurs * /
/ * En mode hte USB, si le bit 5 est 1, il y a un priphrique USB dans le port USB, et si le bit 5 est 0, il n'y a pas de priphrique USB dans le port USB * /
/ * Le bit 4 est utilis pour indiquer l'tat de verrouillage du tampon en mode priphrique USB. Le bit 4 est 1 signifie que le tampon USB est verrouill et le bit 6 est 1 signifie qu'il a t libr * /
/ * Autres bits, rservs, veuillez ne pas modifier * /
#define VAR_RETRY_TIMES 0x25 / * Nombre de tentatives d'opration de transaction USB * /
/ * Si le bit 7 est 0, il ne ressayera pas lorsque NAK est reu, le bit 7 est 1 et le bit 6 est 0, et ressayera  l'infini lors de la rception de NAK (vous pouvez utiliser la commande CMD_ABORT_NAK pour abandonner la nouvelle tentative), le bit 7 est 1 et 6 est 1 Ressayez au plus 3 secondes lors de la rception de NAK * /
/ * Bit 5 ~ Bit 0 reprsente le nombre de tentatives aprs l'expiration du dlai * /
#define VAR_FILE_BIT_FLAG 0x26 / * Indicateur de bit en mode fichier hte * /
/ * Bit 1 et Bit 0, l'indicateur du systme de fichiers FAT du disque logique, 00-FAT12, 01-FAT16, 10-FAT32, 11-illgal * /
/ * Bit 2, indique si les donnes de la table FAT dans le tampon courant ont t modifies, 0-non modifi, 1-modifi * /
/ * Bit 3, la longueur du fichier doit tre modifie, le fichier actuel est ajout avec les donnes, 0-Aucune modification n'est pas ajoute, 1-Ajout et doit tre modifi * /
/ * Autres bits, rservs, veuillez ne pas modifier * /

/ * Bit 1, reconnaissance automatique, carte 0-SD, carte 1-MMC * /

/ * Bit 4, dlai d'expiration de la commande ACMD41 * /
/ * Bit 5, dlai d'expiration de la commande CMD1 * /
/ * Bit 6, dlai d'expiration de la commande CMD58 * /
/ * Autres bits, rservs, veuillez ne pas modifier * /
